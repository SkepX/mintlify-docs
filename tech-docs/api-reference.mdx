## API & Programmatic Access

Ascend exposes **protocol interfaces** for frontends, bots, and institutional traders.

---

## Interface Categories

| Type | Description |
|------|-------------|
| Read | Markets, prices, funding rates, events |
| Write | Orders, transfers, margin actions |

---

## API Keys

- API keys are issued at the **Trading Account level**
- Trade-only permissions by default
- Each API key is bound to a signing authority

---

## Authentication

- API keys authenticate sessions
- Session tokens are used for request authorization

---

## Transaction Authorization

- All actions are cryptographically signed
- Signatures are validated inside execution circuits
- Unauthorized or replayed actions are rejected
